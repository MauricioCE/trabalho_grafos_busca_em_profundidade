# Projeto Grafos - Algoritmos do menor caminho

Trabalho acadêmico cujo resultado final do desenvolvimento é uma aplicação web que mostre, de maneira interativa e visual, a execução de alguns algoritmos de busca do menor caminho.

A inspiração veio do site [Introduction to the A\* Algorithm](https://www.redblobgames.com/pathfinding/a-star/introduction.html), desenvolvido pela Red Blob Games.

**Algoritmos implementados até o momento:**

1. [x] Bread First Search (BFS) ou Busca em Largura

2. [ ] Depth-First Search (DFS) ou Busca em Profundidade

3. [ ] Algoritmo de Dijkstra
